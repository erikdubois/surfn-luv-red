# Surfn Luv Red icon theme

A variant of the [Surfn](https://github.com/erikdubois/surfn) icon theme. Ships only the
**Surfn-Luv-Red** theme; it inherits the base **Surfn** set, so `surfn-icons-git` is required.

## Installation

```
sudo pacman -S surfn-luv-red-icons-git
```

`surfn-icons-git` (base Surfn) is pulled in automatically as a dependency.

## Manual

Copy `usr/share/icons/Surfn-Luv-Red` into `~/.icons` (or `~/.local/share/icons` on Plasma),
ensure the base `Surfn` theme is present, then select **Surfn Luv Red** in your appearance settings.
